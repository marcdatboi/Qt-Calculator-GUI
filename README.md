



### Overview
* This project is a calculator GUI made with Qt6 C++.



### Prior Knowledge (VSCode)

* This program was developed using **CLion & VSCode**
* Ensure you have the **Qt Extension Pack** from the extensions marketplace if you use VSCode



## Requirements
* CMake version >=3.16
* C++ 17
* Qt 6.10.2 -> Widgets, Gui, Core, Qml
* Ninja CMake Generator
